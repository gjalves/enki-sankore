O software Enki-Sankoré usa os tocadores nativos por padrão no computador para interpretar áudio, vídeo ou mídias flash. No entanto, nem todos os formatos poderão ser tocados se o computador não tiver os codificadores necessários instalado.

Codificadores são pequenos programas que permitem decodificar os vários formatos disponíveis de áudio e vídeo.

Os tocadores usados pelo Enki-Sankoré são :

  • Windows Media Player para Windows, 
  • QuickTime Player para Mac,
  • Phonon (uma abstração do gstreamer ou xine) para Linux.

Você deverá dar preferência ao uso dos seguintes formatos, que deverão ser facilmente decodificados por todos os sistemas :

  • imagens nos formatos .png, .svg, .jpg ou .gif
  • vídeos nos formatos .mp4 ou .flv
  • áudios no formato .mp3

Existem alguns pacotes de codificadores que permitem extender o espectro de formatos multimídia suportados pelo leitor usado pelo Enki-Sankoré.

Seja no entanto cuidadoso quando instalá-los, particularmente pacotes livres que supostamente contém codificadores de várias empresas. Alguns destes pacotes poderão criar problemas ou instabilidades em seu sistema operacional.

Por favor consulte a documentação de seu sistema operacional e o fórum online de suporte.
